# pow()

Returns a result of raising to power

**Syntax**

`pow(`*base*`,` *exponent* `)`

**Arguments**

* *base*: Base value.
* *exponent*: Exponent value.

**Returns**

Returns base raised to the power exponent: base ^ exponent.
