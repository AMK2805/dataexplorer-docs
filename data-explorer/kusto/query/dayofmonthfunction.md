# dayofmonth()

Returns the integer number representing the day number of the given month

    dayofmonth(datetime(2015-12-14)) == 14

**Syntax**

`dayofmonth(`*a_date*`)`

**Arguments**

* `a_date`: A `datetime`.

**Returns**

`day number` of the given month.
