# base64_encodestring()

Encodes a string as base64 string

**Syntax**

`base64_encodestring(`*String*`)`

**Arguments**

* *String*: Input string to be encoded as base64 string.


**Returns**

Returns the string encoded as base64 string.

**Example**

```kusto
print Quine=base64_encodestring("Kusto")
```

|Quine   |
|--------|
|S3VzdG8=|
